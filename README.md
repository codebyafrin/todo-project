# TaskFlow – Full Stack To-Do Application

## 📌 Overview

TaskFlow is a modern full-stack To-Do application designed to help users organize and manage daily tasks efficiently. The application provides secure user authentication and task management features through a clean and responsive user interface.

## ✨ Features

* User Registration and Login
* Secure Authentication using JWT
* Password Encryption using bcrypt
* Create New Tasks
* View Existing Tasks
* Update Task Details
* Delete Tasks
* Responsive User Interface
* RESTful API Architecture
* Express.js Backend
* JSON-based Data Storage

## 🛠️ Technologies Used

### Frontend

* HTML5
* CSS3
* JavaScript

### Backend

* Node.js
* Express.js

### Security

* JSON Web Token (JWT)
* bcrypt.js

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/taskflow-todo.git
```

2. Navigate to backend folder

```bash
cd backend
```

3. Install dependencies

```bash
npm install
```

4. Start the server

```bash
npm start
```

5. Open browser

```text
http://localhost:5000
```
